== Description ==
Normally for creating custom post types and custom taxonomies one has to program accordingly and create and manage these things. This plugin let user to create custom post type and custom taxonomies with the help of well sophisticated user interface.

Custom post type can be asscosiated with the taxonomy while creating them and vice-versa. User can just create custom post types and custom taxonomies in few just few clicks. Very Simple ~!

== Installation ==
Install CPT CT Manager plugin from the 'Plugins' section in your dashboard (Plugins > Add New > Search for CPT CT Manager).

Place the downloaded plugin directory into your wordpress plugin directory.

Activate it through the 'Plugins' section.

== Frequently Asked Questions ==

= Can I associate custom post types to already existing custom taxonomies ? =

Absoloutely. This plugin allows you such association with a great flexibility

= Can I associate custom taxonomies to already existing custom post types ? =

Yes, you can.

= Can I de-associate these things with each other =

No. This functionality will be in next version of plugin.
